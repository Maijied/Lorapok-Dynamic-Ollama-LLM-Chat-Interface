SYSTEM REQUIREMENTS

DESKTOP APPLICATION

OS Windows 10+, macOS 12+, Ubuntu 22.04+
CPU Dual Core 2.0GHz (Quad Core 3.0GHz+ Recommended)
RAM 4 GB (8 GB+ Recommended)
Storage 2 GB free space

PYTHON PACKAGE

Python v3.8 or higher (3.11+ Recommended)
Packages requests, rich, psutil
Ollama v0.1.0 or higher
Network Local network access

COMMAND SYSTEM

MODEL MANAGEMENT

/model Switch to different model
/models List available models
/pull Download new model
/remove Remove installed model

CONVERSATION

/history [limit] Show chat history
/search Search conversations
/save [filename] Save conversation
/clear Clear history

MONITORING

/stats Show statistics
/bench [model] Benchmark performance
/sysinfo System information
/config Show configuration
